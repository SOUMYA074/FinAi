Firebase setup guide
1
Create Firebase project
Go to firebase.google.com → Add project → name it finai-app → disable analytics → Create.
2
Enable Google Authentication
Firebase console → Build → Authentication → Get started → Google → Enable → Save.
3
Create Firestore Database
Build → Firestore Database → Create database → Test mode → Region: asia-south1 → Done.
4
Paste your config into this file
Project settings → Your apps → web icon → Register → copy firebaseConfig → in VS Code press Ctrl+H → find PASTE_YOUR_API_KEY_HERE → replace all 6 values.
5
Deploy to GitHub Pages & add domain
Upload index.html to GitHub → Settings → Pages → Deploy. Then in Firebase → Authentication → Settings → Authorized domains → add yourusername.github.io
