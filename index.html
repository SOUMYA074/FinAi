Firebase setup guide
1
Create Firebase project
firebase.google.com → Add project → finai-app → Create.
2
Enable Google Auth
Build → Authentication → Get started → Google → Enable → Save.
3
Create Firestore
Build → Firestore → Create database → Test mode → asia-south1.
4
Deploy to GitHub Pages
Copy firebaseConfig → paste in index.html → push to GitHub Pages → add domain to Firebase Authorized domains.
